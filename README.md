## ⚙️ Stato del Progetto & Requisiti di Esecuzione

* **Status:** Progetto legacy stabile (R&D). Lo sviluppo è stato intenzionalmente concluso e l'applicazione rimossa dal Google Play Store per assenza di piani di monetizzazione commerciali.
* **Target Environment:** Compilabile esclusivamente tramite **Android Studio 4.x** con **NDK r19c** (Target API 28 - Android 9).
* **Retrocompatibilità:** Per preservare l'infrastruttura originale senza alterare la logica algoritmica, l'applicazione può essere installata, testata ed eseguita in ambiente virtualizzato tramite l'emulatore integrato in Android Studio.
